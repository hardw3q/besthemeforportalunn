# Dark UNN Chrome extension

1. Откройте chrome://extensions и включите «Режим разработчика».
2. Нажмите «Загрузить распакованное» и выберите папку chrome-extension в этом репозитории.
3. Обновите https://portal.unn.ru/ — тема подтянется автоматически.

Файлы:
- manifest.json — описание расширения (Manifest V3).
- content.js — контент-скрипт с темой (добавляет стили на document_start).
- background.js — обработчик клика по иконке и хранение состояния в storage.
- icons/ — активная (оранжевая) и неактивная (#0072BC) иконки с буквой P.

Если меняете script, обновите content.js и в Chrome нажмите «Обновить» у расширения.

Переключение темы:
- В меню расширений появится иконка. Оранжевая «P» — тема активна, синяя (#0072BC) «P» — тема выключена.
- Клик по иконке включает/выключает тему. При выключении страница перезагрузится, чтобы вернуть стандартные стили.

## Установка из запакованного архива (.zip)

1. Скачайте архив с папкой `chrome-extension` и распакуйте его на диск (или упакуйте сами эту папку в zip).
2. Откройте Chrome → `chrome://extensions`.
3. Включите «Режим разработчика» (переключатель справа сверху).
4. Нажмите «Загрузить распакованное» и выберите распакованную папку расширения **или** просто перетащите zip-файл в окно расширений (Chrome сам распакует).
5. Убедитесь, что расширение включено. При необходимости нажмите «Обновить».
6. Откройте `https://portal.unn.ru/` — тема применится автоматически (оранжевая иконка «P» означает активное состояние).

## Установка из CRX

1. Подготовьте CRX-файл расширения (или возьмите готовый).
2. Откройте Chrome → `chrome://extensions`.
3. Включите «Режим разработчика» (переключатель справа сверху).
4. Перетащите CRX-файл в окно `chrome://extensions` и подтвердите установку.
5. Убедитесь, что расширение включено. При необходимости нажмите «Обновить».
6. Откройте `https://portal.unn.ru/` — тема применится автоматически (оранжевая иконка «P» означает активное состояние).
